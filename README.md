### Build & Package

#### Build production JS (Vite)

```
npm run build
```

#### Generate TypeScript declaration files (.d.ts)

```
npm run build:types
```

#### 打包成 npm 可發佈的 tarball

```
npm pack
```

### Generate typedoc

#### Generate docs for all TypeScript files under src

```
npm run gendoc
```
